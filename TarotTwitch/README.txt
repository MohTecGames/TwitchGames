========================TWITCH MYSTICAL TAROT OVERLAY====================================

====================================
======DESCRIPTION:==================
====================================
An interactive, automated Tarot reading experience designed for Twitch streamers. This overlay integrates directly with Twitch chat, allowing viewers to request personalized 3-card readings (Past, Present, and Future) using specific topic commands.

====================================
======FEATURES:=====================
====================================
Automated 3-Card Spreads (Past, Present, and Future). Topic-Specific Readings: Love, Career, Money, Health, Family, and Higher Guidance. Twitch Integration: Powered by Comfy.JS for seamless chat.

====================================
======COOLDOWN & VISUALS:===========
====================================
Cooldown System: Global Cooldown: Prevents stream-wide spam. Personal Cooldown: Prevents individual user spam. Visual Animations: 3D card flipping and text reveal effects.

====================================
======SETUP:=======================
====================================
Ensure all card images are in the "Cards/" folder. Update 'TWITCH_CHANNEL' in the script with your name.

====================================
======OBS INTEGRATION:==============
====================================
Add a "Browser Source" in OBS. Point the source to this HTML file.

====================================
======TRIGGERING A READING:=========
====================================
Viewers type a command followed by their question. Example: "tarotlove What does my heart desire?"

====================================
======COMMAND LIST:=================
====================================
tarotdeck, tarotbeginnings, tarotcareer, tarotmoney, tarotlove, tarotfriendsfamily, tarothealth, tarothigherguidance
